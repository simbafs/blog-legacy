# Vim Auto Comment Plugin

## 套件

[nerdcommenter](https://github.com/preservim/nerdcommenter)

## 好用的指令

在每次動作前都要按 `<leader>`，預設是 反斜線 `\`

## `c<space>`

簡單來說就是 toggle 你的註解

## `cA`

跳到行尾並在行尾加上註解
