# 好用的插件

因為 hexo 本身沒有提供編輯器  
所以每次在編輯文章時都需要打路徑  
這個插件他會去找到你的文章  
然後給你選要編輯那一個  
這只是其中一個功能  
還有**重新命名**、**刪除**、**隔離**和**加入**  
**隔離**是將其他文章暫時隔離，**加入**是將文章加回來
